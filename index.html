Introduction

This post will present how an MLP can be built using analog electronics in an attempt to speed up the forward pass. It also shows how the analog circuit can be modelled in TensorFlow to train the network and find the component values. As a simple example, the network is trained on the XOR problem. You can find the code here.

Although I try to explain everything in great detail, I also assume that you are familiar with analog electronics (in particular op-amps and their transfer functions), reffig{mlp} linear algebra and MLPs.

The Perceptron

In this section I'll briefly review the perceptron and describe ways to model it using op-amps. There are three main operations that happen in the perceptron. First, all inputs are multiplied by weights. refbib{odena2019open,ref2} These weighted inputs are then summed together and lastly passed through an activation function that either squishes the signal from an infinite to a finite space or rectifies it. For the following analysis it will be useful to think of all the values within the network as voltages. refbib{asd} refbib{asd}

Weighting and Summing

The first two operations can be easily achieved with a non-inverting summing amplifier, whose transfer function exactly matches that of a perceptron.
